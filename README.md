# Student Record Management System

## Overview

A menu-driven Student Record Management System developed in C using a Singly Linked List and modular programming approach. The application allows users to manage student records efficiently through operations such as add, delete, modify, search, sort, and file-based data persistence.

## Features

- Add new student records
- Automatic roll number generation
- Delete records by roll number or name
- Modify student information
- Display all records
- Sort records by name or percentage
- Reverse the linked list
- Delete all records
- Save records to file
- Load records automatically at startup

## Technologies Used

- C Programming
- Data Structures (Singly Linked List)
- Dynamic Memory Allocation
- File Handling
- Makefile
- GCC
- Linux
